## 📌 Sobre o Projeto

Este projeto consiste em um gerador de QR Code para e-commerce desenvolvido utilizando Node.js.

A aplicação foi criada com o objetivo de praticar:

- Estruturação de projetos backend
- Organização de código por camadas
- Uso de dependências externas
- Manipulação de pacotes NPM
- Geração dinâmica de QR Codes
